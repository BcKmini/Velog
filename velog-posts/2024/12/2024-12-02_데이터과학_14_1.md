## ggplot2
- ggplot()함수는 앞에서 설명한 고수준, 저수준 작도를 편리하게 사용할 수 있도록 개발된 패키지

- Advantages of ggplot2
consistent underlying grammar of graphics (Wilkinson, 2005)
plot specification at a high level of abstraction
very flexible
theme system for polishing plot appearance
mature and complete graphics system
many users, active mailing list

&gt; ### 가본 syntax
library(ggplot2)
ggplot(df, aes(x=x축 데이터, y=y축 데이터)) + geom_함수
- In ggplot land aesthetic means "something you can see".
  - geom_함수
geom_bar()
geom_line()
geom_point() 
geom_histogram()
labs()
facet_wrap()

```py
data &lt;- read.csv("../mydata/경기도 화성시_관광통계_20231025.csv",fileEncoding="CP949")
```

```py
# ggplot 객체 생성
p &lt;- ggplot(data, aes(x = 번호, y = 공룡알화석산지))

p &lt;- p + geom_point(fill = 'red', color = 'blue', shape = 21, size = 4)

# 최종 플롯 출력
p + geom_line(color='blue', linetype=2)
```

## 📌 막대그래프 - 평균을 높이로, 수직누적 막대그래프- 두 그룹별 평균

- 그룹별 평균을 구할 때 tapply보다는 summaryBy를 이용(데이터프레임 필요)
- aes(변수 1개)
- 막대그래프의 y축은 기본적으로 주어진 변수의 count(빈도수)
- 예제처럼  cyl에 따른 mpg의 평균을 높이로 지정할 때는 두 변수를 모두 지정하고, 옵션stat=‘identity’ 지정
- 데이터 mtcars에서 두 변수 cyl과 gear의 각 그룹별 평균 구하기
- geom_bar()
- stat=‘identity’ 옵션 사용

![](https://velog.velcdn.com/images/mi_nini/post/f940b578-771a-4ed6-965b-8e74b9d95d94/image.png) | ![](https://velog.velcdn.com/images/mi_nini/post/86c17dc5-ee8b-40a2-b646-c289909cc9b8/image.png)
--| --|

```py
library(doBy)
tbl &lt;- summaryBy(mpg ~ cyl, data=mtcars, FUN=mean)
tbl

p &lt;- ggplot(tbl, aes(factor(cyl), mpg.mean))
p + geom_bar(stat='identity', fill='gold', color='red')
```
```py
library(doBy)
tbl &lt;- summaryBy(mpg ~ cyl+gear, data=mtcars, FUN=mean)
tbl

p &lt;- ggplot(tbl, aes(factor(cyl), mpg.mean, fill=factor(gear)))
p + geom_bar(stat='identity')
```

# 📌 시계열 그래프_ geom_line()

## 꺽은선 그래프 – no 옵션
- aes(x, y) 순서로
- geom_line(): 선추가
- geom_point(): 점추가
- color는 색상
- shape: 점모양
- size: 점 크기
- linetype: 선모양
- linewidth:선 굵기



![](https://velog.velcdn.com/images/mi_nini/post/602c5d2c-e6c3-4a42-95b2-4f3230ac6e20/image.png) | ![](https://velog.velcdn.com/images/mi_nini/post/523d81d5-6f2f-4977-be1d-ac5fb826bf07/image.png)

&gt;## 점모양
shape = 번호 (plot에서는 pch)
![](https://velog.velcdn.com/images/mi_nini/post/02ede2e6-c450-4ae7-9e6f-e709610d2d05/image.png)

&gt; 선 모양: plot에서는 lty, ggplot에서는 linetype

&gt;| 인 수      | 설명                 |
|------------|----------------------|
| `linetype=0` | 투명선              |
| `linetype=1` | 실선                |
| `linetype=2` | 대시선              |
| `linetype=3` | 점선                |
| `linetype=4` | 점선과 대시선       |
| `linetype=5` | 긴 대시선           |
| `linetype=6` | 2개의 대시선        |
