# 📌 들어가기 전에

### 졸업작품 전시회

  1. 일시: 2024. 11. 13.(수) 10:00 ~ 18:00
  2. 장소: 교수연구동 컨벤션홀



우리 학교의 경우 졸업을 위해서는 반드시 졸업작품을 제출하지 않아도 되며, 대체요건을 충족하여 졸업할 수 있다. 2024년 5월 16일 개정된 [졸업작품 규정](https://cs.kku.ac.kr/noticeView.do?siteId=CS&boardSeq=2095&menuSeq=8848&searchBy=title&searchValue=&categorySeq=&curBoardDispType=LIST&curPage=12&pageNum=1&seq=32703)에 따르면, 졸업을 위한 대체요건은 아래와 같다.

* * *

**구분**| **졸업논문 연구형**| **졸업작품 일반형**| **졸업작품 대체형**  
---|---|---|---  
**선택 시기**|  4학년 1학기 개강 전| 4학년 1학기 개강 전| 4학년 1학기 개강 전  
**지도교수 배정**|  개별 상담 후 배정| 학과에서 일괄 배정| 학과에서 일괄 배정  
**필수 교과목**|  소프트웨어종합설계, 프로젝트종합설계| 소프트웨어종합설계, 프로젝트종합설계| 없음  
**중간심사**|  필수| 필수| 없음  
**최종심사**|  필수| 필수| 대체요건 충족 시 면제  
**전시회 참여**|  필수| 필수| 면제  
**대체요건**|  해당 없음| 해당 없음| 아래 요건 중 하나 충족 필요  
**대체요건 종류**|  -| -| ① 취업 (4대 보험 가입 필수)  
| | | ② 자격증 + 공모전 입상  
| | | ③ 자격증 + 인턴십  
| | | ④ 공모전 입상 + 인턴십  
| | | ⑤ 창업 (사업자등록 필수)  
**기타 사항**|  미참여 시 F 성적 부여 가능| 미참여 시 F 성적 부여 가능| 대체요건 관련 증빙서류 제출 필수  
  
나는 학생회에 소속되어있기에 전날 컴퓨터 셋팅도 하고 선배들이 한 작품 설명에 대해서 내가 셋팅하면서 먼저 볼 수 있어서 좋았다. 다양한 작품들이 많이 통과되어 약 50개의 작품들이 있었다.

> # 요약🚀
> 
>   1. 학점이 중요하지는 않지만 기본 이상은 해야한다.
>   2. 대학원을 가기전 교수님을 너무 믿지 말아야한다.
>   3. 대외활동과 학교 내 많은 행상에 참여해라.
>   4. 기회가 왔을때 잡을 수 있는 준비는 자신이 항상 해야한다.
>   5. 학교 동기, 선(후)배와 친하게 지내라. (후에 도움이 될 수 있을 것이다.)
